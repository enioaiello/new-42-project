# New 42 project

L'essentiel pour débuter un nouveau projet chez 42 Paris (piscine).

## Utilisation

Vous avez **deux méthodes** pour utiliser le modèle.

> [!IMPORTANT]
> Ce projet n'a pas pour but d'être utilisé :
> - Pendant les examens
> - Pendant les rushs
> 
> **Vous utilisez ce contenu à vos risques !**

1. [Méthode automatique](#automatique)
2. [Méthode manuelle](#manuelle)

### Automatique

1. Téléchargez le fichier `Shell`present [dans la branche concernée](https://github.com/enioaiello/new-42-project/tree/automatic).

2. Rendez le fichier exécutable

```bash
chmod +x create-my-env
```

3. Exécutez le fichier

```bash
./create-my-env
```

4. Attendez la création de votre environnement de travail

5. À vous de jouer !

### Manuelle

1. Téléchargez le dépôt au format ZIP [depuis la branche concernee](https://github.com/enioaiello/new-42-project/tree/manual)

2. Décompressez le fichier

```bash
unzip ~/Downloads/new-42-project-manual.zip
```

3. Déplacez le contenu du dossier dans votre projet

```bash
mv -r ~/Downloads/new-42-project-manual/* # chemin de votre projet
```

4. May the force be with you!
